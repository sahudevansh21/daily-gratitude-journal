# Simple Daily Gratitude Journal

## Problem
Cultivating a daily gratitude practice can improve mental well-being, but many digital solutions are overly complicated, require accounts, or come with subscription fees. Users need a simple, private, and free tool to record their daily reflections without unnecessary friction.

## Solution
This website offers a minimalist interface for users to quickly jot down what they're grateful for each day. Entries are stored securely in the browser's local storage, ensuring privacy and accessibility at any time, allowing for a consistent and uncomplicated journaling experience.

## Pages
*   Today's Entry
*   Calendar View
*   Past Entries

## Technologies
*   Next.js 14 App Router
*   React 18
*   Client-side local storage for data persistence
*   Pure CSS for stunning, glassmorphic design

## Setup and Running
1.  **Clone the repository:** (If this were a real repo)
    ```bash
    git clone <repository-url>
    cd daily-gratitude-journal
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run the development server:**
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) with your browser.

4.  **Build for production:**
    ```bash
    npm run build
    ```

5.  **Start the production server:**
    ```bash
    npm run start
    ```
